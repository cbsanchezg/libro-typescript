# TypeScript - Ruta de Aprendizaje
Este libro es una guía accesible para aprender TypeScript, ideal para principiantes y programadores con experiencia en JavaScript. Incluye ejemplos prácticos explicados en detalle, sin necesidad de usar una computadora, y en este repositorio ofrecemos todos los ejemplos utilizados en cada capítulo del libro.
